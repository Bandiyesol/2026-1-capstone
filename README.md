# 🏛️ The Last Rune: 잊혀진 신들의 설계도

> **"세계의 법칙을 재구성하라. 당신은 마지막 기록이자 희망이다."**

공허(The Void)에 잠식된 세계를 되돌리기 위해
고대 신의 언어 **'룬(Rune)'**을 활용해 인과율을 재설계하는
**탑다운 로그라이크 액션 게임**입니다.

---

## 🌌 스토리 라인 (Storyline)

고대 신의 언어 ‘룬’으로 설계된 세계는
어느 날 나타난 **공허(The Void)**에 의해 7개의 성계로 분열됩니다.

모든 존재를 무로 되돌리려는 시스템,
**'무명의 관찰자'**가 최종 보스로 등장하고,

플레이어는 심장에 새겨진 **The Last Rune**을 통해
붕괴된 세계의 법칙을 다시 구축해야 합니다.

---

## 🎮 게임 특징 (Core Features)

### 🧩 로직 카드 (Logic Card) & 룬 시스템

단순한 능력치 상승이 아닌,
**공격의 구조 자체를 설계하는 시스템**

* **발사 단계**: 멀티샷, 크기 증가, 속도 증가
* **이동 단계**: 유도, 관통, 도탄, 회전 궤도
* **충돌 단계**: 폭발, 장판 생성, 연쇄 피해

👉 플레이 스타일을 직접 “코딩하듯” 조합하는 핵심 메커니즘

---

### ⚔️ 무기 & 아이템 위계

* **무기 (10종)**
  소드 / 보우 / 스태프 / 망치 / 마도서 / 부메랑 / 오브 / 낫 / 총 등

* **아이템 등급**

  ```
  일반 → 희귀 → 유니크 → 전설 → 성물(Relic)
  ```

* **성물 (Relic)**

  * 총 12종
  * 캐릭터 플레이 방식 자체를 변화시키는 핵심 요소

---

## 🗺️ 세계관: 7개의 바이옴 (7 Biomes)

공허에 의해 분열된 7개의 성계
각각 고유한 기믹과 분위기를 가진 스테이지 구성

(숲,초원), 동굴, 바다, 용암, 설원, 사막, 차원의 틈(보이드)

각각의 바이옴은 환경 기믹, 수호자가 존재함

## 💻 기술 스택 및 구조 (Tech Stack)
